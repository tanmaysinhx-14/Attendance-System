# Attendance System – Feature Update

This repository contains a **feature update for https://careerinstitute.co.in**.

This project aims to implement a **comprehensive attendance viewing system for students and administrators**, providing a **clear, structured, and intuitive representation of attendance data**. The module focuses on **month-wise visualization, real-time statistics, and responsive calendar-based layouts**, enabling efficient tracking, analysis, and administrative oversight with minimal effort.

---

## 📃Project Overview (Attendance Viewer 👀)

This sub-feature of attendance system enables:


- **Month-wise Attendance View (Students)**  
  Students can view their attendance records organized by month with clear date-wise representation.

- **Responsive Calendar Layout**  
  Optimized calendar-based interface that adapts seamlessly across desktop, tablet, and mobile devices.

- **Dynamic Attendance Statistics**  
  Real-time calculation of total days present, total days absent, and overall attendance percentage.

- **Multi-Batch Attendance Preview (Admins)**  
  Administrators can preview attendance data for multiple batches simultaneously with a single action.


The solution is designed to be efficient, scalable, and user-friendly for academic institutions.

---

## 🌐 Tech Stack

This project is built using:
- **HTML** – Frontend framework
- **Javascript** – Dynamic Calendar Element
- **PHP** – Server and Backend Processing
- **CSS 3** – Responsive UI styling
